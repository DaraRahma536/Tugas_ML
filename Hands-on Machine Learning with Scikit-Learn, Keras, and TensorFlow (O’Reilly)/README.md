# Hands-on-Machine-Learning

### Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow: Concepts, Tools, and Techniques to Build Intelligent Systems
Penulis: Aurélien Géron  
Edisi: Kedua (2019)

**Tujuan Buku:**  
Buku ini bertujuan untuk membekali pembaca dengan pemahaman konseptual dan keterampilan praktis untuk membangun sistem Machine Learning menggunakan pustaka Python populer seperti Scikit-Learn, Keras, dan TensorFlow. Buku ini dirancang untuk pemula yang belum memiliki pengetahuan mendalam tentang ML, dengan pendekatan hands-on melalui contoh nyata dan sedikit teori.

**Struktur Buku:**  
Bagian I: Fundamentals of Machine Learning  
Membahas dasar-dasar ML, algoritma klasik, dan teknik pra-pemrosesan data.  

Bab 1: The Machine Learning Landscape  
Pengenalan ML, jenis-jenis sistem (supervised, unsupervised, semi-supervised, reinforcement learning), tantangan utama (overfitting, underfitting, kualitas data), dan evaluasi model.  

Bab 2: End-to-End Machine Learning Project  

Studi kasus lengkap proyek ML: dari pengumpulan data, eksplorasi, pra-pemrosesan, pemilihan model, hingga deployment.  

Bab 3: Classification  

Teknik klasifikasi biner dan multi-kelas, metrik evaluasi (precision, recall, ROC curve), dan analisis kesalahan.  

Bab 4–9: Algoritma dan Teknik Lanjutan  

Regresi Linear, SVM, Decision Trees, Ensemble Methods, Dimensionality Reduction (PCA), dan teknik unsupervised learning (clustering, anomaly detection).  

Bagian II: Neural Networks and Deep Learning  
Fokus pada jaringan saraf tiruan dan penerapan dengan TensorFlow dan Keras.

Bab 10: Introduction to Artificial Neural Networks with Keras  

Dasar ANN, implementasi dengan Keras, dan hyperparameter tuning.  

Bab 11: Pelatihan Jaringan yang Mendalam  

Teknik training deep nets (regularisasi, optimizers, batch normalization), custom model dengan TensorFlow, dan pra-pemrosesan data skala besar.  

**Target Pembaca:**  
* Pemula yang ingin belajar ML dari nol.  
* Praktisi yang ingin memperdalam penggunaan Scikit-Learn, Keras, dan TensorFlow.  
* Mahasiswa, peneliti, atau engineer yang ingin membangun sistem ML yang siap produksi.  

### Kesimpulan:
Buku ini merupakan panduan komprehensif yang menggabungkan teori dan praktik dalam dunia Machine Learning modern. Dengan fokus pada alat-alat produksi seperti Scikit-Learn, Keras, dan TensorFlow, buku ini cocok bagi siapa saja yang ingin menguasai ML secara mendalam dan aplikatif.
